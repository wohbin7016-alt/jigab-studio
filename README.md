# 지갑스튜디오

시니어 유튜브 AI 영상 자동 생성 도구.

## 사용 방법

https://wohbin7016-alt.github.io/jigab-studio/ 접속 → 본인 Google API 키 등록 → 주제 선택 → 영상 생성

## 기능

- AI 대본 (Google Gemini)
- AI 이미지 (Imagen 4.0)
- AI 음성 (ElevenLabs 선택)
- 자동 자막 (SRT)
- CapCut 호환 zip 패키지

## 비용

영상 1개당 약 1,000~1,500원 (사용자 Google 계정 결제)
