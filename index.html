📋 CapCut 설치 방법

  1. 받은 zip을 바탕화면에 압축 풀기
  2. CapCut 앱에서 "가져오기(Import)" 클릭
  3. 압축 푼 폴더의 narration.mp3와 이미지들을 선택해 타임라인에 끌어다 놓기
  4. 자막은 subtitles.srt 파일 드래그

🔑 API 키 설정

영상 생성에 필요합니다. 본인 Google 계정으로 결제돼요.

1. Google Gemini 키 받기 (필수)

aistudio.google.com/apikey

로그인 → Create API key → 복사

2. ElevenLabs 키 (선택, 한국어 고품질 TTS)

elevenlabs.io 설정
